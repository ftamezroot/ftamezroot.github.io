IT Asset

Application for managing IT assets. In the front-end, tables are displayed with information about assets, users that are synchronized by batch (Python) loading daily from the company's Active Directory. It also has dashboards to show equipment that is out of warranty, as well as annual budget planning for equipment replacement. Another functionality is that QR codes are generated so that security personnel can verify if the equipment is assigned to a certain user and if they have permission to take it outside the plant, from a tablet.

K8S + Docker + NGINX + Django + Gunicorn + Bootstrap + Jquery.
